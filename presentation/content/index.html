Git in Daily Works

Git Commands

Git is composed of over hundred seperate programm as its entire tool set. It looks horrifying at first glance.

Git Commands

Fortunately, core commands set of git is much more smaller.

basic:
    add, commit, diff, checkout, check, pull, push, rebase, merge,
    log, status, init, clone 

adavnced:
    bitsect, stash, blame ...

Initiate you workspace

  • Create your own repository

    $mkdir Cord
$cd Cord
$git init

  • Or clone from another repository

    $git clone <repo>

Stage your changes

  • Use git add to put new file (track it) and changes into index

    $touch README
$git add README

  • Shortcuts

    #Recurisve add all files in the directory
$git add .
#Add all tracked files
$git add -u

View work tree status

Use git status to view your staged or unstaged changes

Ignore files

  • Put .gitignore at root of your repositroy

    $ls -a
.  ..  deploy  .git  .gitignore  .gitmodules  src  utils

    You can add it to your repositroy

  • Use wildcards in .gitignore to exclude files you don't want to track

    $cat .gitignore 
*.pyc
*~
*swp
.*
src/cord/static/
src/cord/media/
